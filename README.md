# Arabica AI Chat 🌍🤖  
A bilingual AI chatbot that seamlessly switches between English and Arabic, using real-time translation and AI-powered responses to enable natural, fluid conversations across languages. Built with a clean Tkinter interface, it enhances accessibility, communication, and language learning.

---

## 🚀 Features  
- 🌐 Bilingual Conversations — Supports both English and Arabic seamlessly  
- 🔁 Real-Time Translation — Automatically translates input and output  
- 🤖 AI-Powered Responses — Context-aware replies using advanced AI  
- 🪟 Clean GUI — Minimal, user-friendly Tkinter interface  
- ⚡ Fast & Responsive — Smooth real-time interaction  

---

## 🧠 Tech Stack  
- 🐍 Python — Core application logic  
- 🪟 Tkinter — Graphical user interface  
- 🤖 Gemini API — AI response generation  
- 🌍 Translation Layer — Real-time language processing  

---

## 💡 How It Works  
1. User enters a message in English or Arabic  
2. System detects and translates input if needed  
3. AI processes the message and generates a response  
4. Output is translated back to the user’s language
